Robotarium Intro

Robotarium is a free multi-agents robots hardware platform launched by Prof. Magnus Egerstedt’s lab from GaTech. It is developed by the intention of bringing the multi-agent hardware platform accessible to everyone, as maintaining it is expensive and not everyone can afford it. One can use up to 20 robots to run the experiments. Robots are based on differential drive hardware. They also have simulators where one can test the experiments before uploading to the Robotatium. They support Matlab and python languages. Check out their Matlab and python simulators.

Experiments can be submitted to run on the Robotarium hardware with a Robotarium account. Once logged in the new experiment tab contains an uploader to submit the experiment code along with some description and instructions to run it. According to their website, they usually perform the experiments in a few days and send the inference data for it.
